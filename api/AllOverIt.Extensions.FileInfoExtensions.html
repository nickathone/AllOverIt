    Parameters
    Type Name Description
    System.IO.FileInfo fileInfo

    The FileInfo instance containing information about a file or directory.
    Returns
    Type Description
    System.String

    Returns the fully qualified path of the directory or file.
    Remarks

    This method returns the equivalent of fileInfo.FullName with the exception that if the length of the string is 260 or more characters the method falls back to using reflection to obtain the string rather than throw a PathTooLongException exception.
